# 🎬 YAHO Project

YAHO Project는 애니메이션 정보를 조회하고 사용자 간 소통이 가능한 웹 플랫폼입니다.

Spring Boot 기반 MVC 구조로 개발되었으며,
애니메이션 일정 조회, 작품 평가, 과거 작품 검색 기능 등을 제공합니다.

JSL 인재개발원 팀 프로젝트로 진행되었습니다.



## 🛠 Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA

### Database
- MySQL
- Oracle

### Frontend
- HTML5
- CSS3
- JavaScript
- Thymeleaf

### DevOps / Tools
- Git
- GitHub
- AWS (RDS)



## 🏗 Architecture

- MVC 패턴 기반 설계
- Controller → Service → Repository 구조
- RESTful API 설계 적용
- DB 이중 구성 (MySQL + Oracle)



## ✨ 주요 기능

### 📅 애니메이션 일정 조회
- 분기별 애니메이션 편성표 제공
- 방송 예정작 필터링 기능

### ⭐ 작품 평가 시스템
- 사용자 평점 등록 기능
- 평균 평점 자동 계산

### 🔍 과거 작품 검색
- 제목 기반 검색
- 상세 페이지 정보 제공



## ⚙️ Environment Setup

본 프로젝트는 보안상의 이유로 실제 DB 설정 파일을 포함하지 않습니다.

1. `application-example.properties` 파일을 복사합니다.
2. 파일명을 `application.properties`로 변경합니다.
3. 로컬 DB 환경에 맞게 설정을 수정합니다.
4. 프로젝트를 실행합니다.
